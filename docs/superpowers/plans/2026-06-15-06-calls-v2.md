# Calls v2 — Transcript / Subtitles / Collapse-Expand / Typing Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a live transcript panel that doubles as subtitles to video calls, with desktop collapse → subtitle overlay, mobile expand → full chat below the video, and the ability to type during a call.

**Architecture:** Wrap the existing `VideoCall.tsx` in a `CallLayout` that hosts the video stage + a `TranscriptPanel`. Real-time captions come from `useSTT` (realtime mode) streaming the local mic transcript, plus incoming peers' transcripts relayed over the existing conversation socket (a new lightweight `conversationCaption` broadcast). Typed messages reuse `conversationMessageCreate` and appear in the same transcript list. Collapse/expand is local UI state with the two layouts from the mockups.

**Tech Stack:** React, ugly-app `useSTT` (`ugly-app/client`), Cloudflare Realtime (existing), vitest.

> Verified facts: `useSTT(socket, {mode:'realtime'})` returns `{start, stop, transcript, isFinal, listening}` and streams `stt:transcript`. `VideoCall.tsx` tracks `CallParticipant` roster + polls `conversation.call`. Visual source: `mockups/call-bot.html`, `mockups/call-2p.html` (transcript panel, `.subs` overlay, `.mcall.expanded`, `.tp-cbox`).

### Task 1: Transcript model + merge reducer (TDD)

**Files:**
- Create: `shared/transcript.ts`
- Test: `tests/unit/transcript.test.ts`

A transcript is an ordered list of turns; realtime STT replaces the in-progress (non-final) turn for a speaker until `isFinal`, then appends. Typed messages append as final turns flagged `typed`.

- [ ] **Step 1: Failing test**

```ts
// tests/unit/transcript.test.ts
import { describe, it, expect } from 'vitest';
import { upsertTurn, type Turn } from '../../shared/transcript';

describe('upsertTurn', () => {
  it('replaces the live (non-final) turn for a speaker, then appends on final', () => {
    let turns: Turn[] = [];
    turns = upsertTurn(turns, { speaker: 'dana', text: 'hel', final: false, at: 1 });
    turns = upsertTurn(turns, { speaker: 'dana', text: 'hello', final: false, at: 2 });
    expect(turns).toHaveLength(1);
    expect(turns[0].text).toBe('hello');
    turns = upsertTurn(turns, { speaker: 'dana', text: 'hello there', final: true, at: 3 });
    expect(turns).toHaveLength(1);
    expect(turns[0].final).toBe(true);
    // next live turn starts a new row
    turns = upsertTurn(turns, { speaker: 'dana', text: 'and', final: false, at: 4 });
    expect(turns).toHaveLength(2);
  });
  it('keeps a typed turn distinct and final', () => {
    let turns: Turn[] = [];
    turns = upsertTurn(turns, { speaker: 'me', text: 'link?', final: true, typed: true, at: 1 });
    expect(turns[0].typed).toBe(true);
  });
});
```

- [ ] **Step 2: Run, expect FAIL.** `npx vitest run tests/unit/transcript.test.ts`

- [ ] **Step 3: Implement**

```ts
// shared/transcript.ts
export interface Turn { speaker: string; text: string; final: boolean; typed?: boolean; at: number }

// If the last turn for this speaker is still live (non-final, not typed), replace it;
// otherwise append a new turn.
export function upsertTurn(turns: Turn[], next: Turn): Turn[] {
  const last = turns[turns.length - 1];
  if (last && last.speaker === next.speaker && !last.final && !last.typed) {
    const copy = turns.slice(0, -1);
    return [...copy, { ...next }];
  }
  return [...turns, { ...next }];
}
```

- [ ] **Step 4: Run, expect PASS.** `npx vitest run tests/unit/transcript.test.ts`

// client/lib/useCallTranscript.ts (shape)
export function useCallTranscript(socket, conversationId, meId, active) {
  const [turns, setTurns] = useState<Turn[]>([]);
  const stt = useSTT(socket, { mode: 'realtime', conversationId });
  useEffect(() => { if (active) void stt.start(); else stt.stop(); }, [active]);
  useEffect(() => {
    if (!stt.transcript) return;
