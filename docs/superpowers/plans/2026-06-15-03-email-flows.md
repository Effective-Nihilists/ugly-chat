**Goal:** Let users start a 1:1 chat by email, create a group by adding email addresses, and manage a conversation in a settings screen — all keyed off **email address** (no usernames/friend-requests).

**Architecture:** Add an email→userId resolver on the server that calls ugly.bot's user-lookup endpoint and falls back to minting a pending invite. **Tech Stack:** TypeScript, Zod, ugly-app server (`getDb`, `uglyBotRequest`/fetch to ugly.bot), React, vitest.

> Visual source of truth: `mockups/new-chat.html`, `mockups/new-group.html`, `mockups/chat-settings.html`, components from `mockups/brand.css` (`.modal`, `.tokens`, `.chip`, `.member`, `.srow`, `.switch`).

```ts
// server/resolveEmail.ts
import { normalizeEmail, isValidEmail } from '../shared/email';

export type ResolveResult =
  | { status: 'found'; userId: string; name: string }
  | { status: 'invite'; email: string };

type Env = { UGLY_BOT_URL?: string; UGLY_BOT_TOKEN?: string };

export async function resolveEmailToUser(
  rawEmail: string,
  env: Env,
  fetchFn: typeof fetch = fetch,
): Promise<ResolveResult> {
  const email = normalizeEmail(rawEmail);
  if (!isValidEmail(email)) throw new Error('invalid email');
  const base = env.UGLY_BOT_URL ?? 'https://ugly.bot';
  const token = env.UGLY_BOT_TOKEN;
  if (!token) throw new Error('UGLY_BOT_TOKEN not set');
  const res = await fetchFn(`${base}/v1/users/email?email=${encodeURIComponent(email)}`, {
    headers: { Authorization: `Bearer ${token}` },
  });
  if (res.ok) {
    const data = (await res.json()) as { user?: { userId?: string; name?: string } };
    if (data.user?.userId) return { status: 'found', userId: data.user.userId, name: data.user.name ?? email };
  }
  // 404 / no user → invite path (the invite email is sent by the create handler)
  return { status: 'invite', email };
}
```

- [ ] **Step 3: Implement handlers** in `server/handlers.ts`. Reuse `resolveEmailToUser`, the existing `conversationCreate` engine call (see `client/lib/bots.ts:startBotChat` for the `conversationCreate` shape), and `conversationMemberAdd`/`engineConversationUserAdd`:

```ts
// imports at top of server/handlers.ts
import { resolveEmailToUser } from './resolveEmail';
import { directConversationId } from '../shared/conversationId';

// inside the requests object:
resolveEmail: async (_userId, input) => resolveEmailToUser(input.email, getEnv()),

conversationCreateDirect: async (userId, input) => {
  const r = await resolveEmailToUser(input.email, getEnv());
  if (r.status === 'invite') {
    await sendInviteEmail(r.email, userId); // Task 4
    return { conversationId: '', invited: true };
  }
  const id = directConversationId(userId, r.userId);
  const existing = await getDb().getDoc(collections.conversation, id);
  if (!existing) {
    await engineConversationCreate({ id, type: 'direct', mode: 'private', ownerIds: [userId, r.userId] }, userId);
    await engineConversationUserAdd({ conversationId: id, userId: r.userId, role: 'member', visibility: 'visible' }, userId);
  }
  return { conversationId: id, invited: false };
},

groupCreate: async (userId, input) => {
  const id = `grp-${userId}-${Date.now().toString(36)}`;
  await engineConversationCreate({ id, type: 'group', title: input.title ?? 'New group', mode: 'private', ownerIds: [userId] }, userId);
  const invited: string[] = [];
  for (const raw of input.emails) {
    const r = await resolveEmailToUser(raw, getEnv());
    if (r.status === 'found') {
      await engineConversationUserAdd({ conversationId: id, userId: r.userId, role: 'member', visibility: 'visible' }, userId);
    } else {
      await sendInviteEmail(r.email, userId, id);
      invited.push(r.email);
    }
  }
  return { conversationId: id, invited };
},
```

Use the same `engineConversationCreate` / `engineConversationUserAdd` imports the file already uses (the engine functions backing `conversationCreate` and `conversationMemberAdd`). `getEnv()` returns `process.env`-equivalent (match how `server/bots.ts` reads `UGLY_BOT_URL`/`UGLY_BOT_TOKEN`).
